SCRIPT SQL (localizado dentro da pasta resources com nome "script.sql") SEJA COPIADO E COLADO NO QUERYTOOL DO POSTGRESQL PARA A CRIAÇÃO DO BANCO CORRETO E DA TABELA USUARIOS.
CASO DESEJE INCLUIR A TABELA USERS DENTRO DE ALGUM BANCO EXISTENTE, É NECESSÁRIO QUE MUDE O NOME DO DATABASE NA CLASSE UserRepository.
